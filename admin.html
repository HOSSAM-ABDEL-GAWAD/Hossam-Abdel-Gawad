بديل: استخدام Google Apps Script (أكثر تحكماً)

إذا أردت إضافة/حذف بيانات مباشرة من لوحة التحكم هذه إلى Google Sheets، انسخ هذا الكود إلى Apps Script:

function doPost(e) { var sheet = SpreadsheetApp.openById('YOUR_SHEET_ID'); var data = JSON.parse(e.postData.contents); var tab = sheet.getSheetByName(data.sheet); tab.appendRow(data.row); return ContentService.createTextOutput( JSON.stringify({success: true}) ).setMimeType(ContentService.MimeType.JSON); } function doGet(e) { var sheet = SpreadsheetApp.openById('YOUR_SHEET_ID'); var tab = sheet.getSheetByName(e.parameter.sheet); var rows = tab.getDataRange().getValues(); return ContentService.createTextOutput( JSON.stringify(rows) ).setMimeType(ContentService.MimeType.JSON); }

انشر الـ Script كـ Web App مع صلاحية "Anyone" ثم ضع رابطه في الحقل أعلاه.
